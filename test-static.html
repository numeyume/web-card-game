🚀 How to Test

💻 For Developers

  1. 1 Run test-all.bat to validate all systems
  2. 2 Run start-server.bat to start the backend
  3. 3 In /client folder, run start-client.bat
  4. 4 Open http://localhost:5173 in browser

🎮 For Testing

  1. 1 Create a room and join with multiple browser tabs
  2. 2 Build custom cards using the Card Builder
  3. 3 Start a game and play cards to test gameplay
  4. 4 End game to see results, analytics, and voting
